# 📖 Overview

CityPulse is a Smart City Analytics Dashboard developed using Microsoft Power BI to demonstrate how Business Intelligence can help city administrators, urban planners, and government organizations make informed decisions using data.

The project consolidates multiple urban datasets into a unified analytics platform, providing interactive dashboards for monitoring traffic, public safety, environmental conditions, and water resource management.

Designed as a portfolio-ready Business Intelligence solution, CityPulse showcases modern dashboard development, data modeling, Power Query transformations, DAX calculations, and interactive reporting using realistic city datasets.

---

# 🎯 Problem Statement

Modern cities generate enormous volumes of operational data across transportation, public safety, utilities, and environmental monitoring.

However, this information is often distributed across different departments, making it difficult to gain a unified view of city performance.

Traditional reporting systems are often:

* Fragmented across departments
* Time-consuming to analyze
* Static and difficult to explore
* Limited in providing actionable insights
* Inefficient for strategic planning
* Challenging for decision-makers

CityPulse addresses these challenges by providing a centralized Power BI dashboard that transforms raw urban data into meaningful insights through interactive visualizations and KPI-driven reporting.
