# Smart City Analytics Dashboard — Power BI Project

A beginner-to-intermediate, interview-ready Power BI portfolio project covering Traffic, Crime,
Air Pollution, and Water Consumption analytics for a simulated metropolitan region, plus a
unified Executive dashboard.

## Start here
1. Read `Docs/SmartCity_PowerBI_Guide.md` — full build guide (data model, Power Query steps,
   DAX measures explained, page-by-page visual instructions, interview Q&A, resume/README text).
2. Open `Preview/SmartCity_Dashboard_Preview.html` in any browser to see a working, interactive
   preview of all 5 pages built from the real sample data (no Power BI required to view it).
3. Open `Data/SmartCity_SampleData.xlsx` in Power BI Desktop (Get Data → Excel) to start building
   the live report.
4. Apply `Theme/LightTheme_iOS.json` via View → Themes → Browse for themes for the light,
   iOS-inspired look.
5. Save your finished report into `PBIX/`.

## Folder structure
```
SmartCity-PowerBI-Project/
├── PBIX/        your .pbix file goes here once built
├── Data/        SmartCity_SampleData.xlsx (3 dimension + 4 fact tables)
├── Theme/       LightTheme_iOS.json
├── Docs/        SmartCity_PowerBI_Guide.md
├── Preview/     SmartCity_Dashboard_Preview.html
└── README.md
```
